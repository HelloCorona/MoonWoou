# MoonWoou

## 개요
> 프로젝트명
``` 
*Hello Corona의 공개용 소스
```

## 설명
> 공개 프로젝트 저장소
```
Hello Corona의 프로젝트의 일환으로
Corona를 처음 접하거나 초보자들을 위한
샘플 소스를 제공하고 스터디를 유도한다
```

## 특이사항
> 업데이트
```
비정기적으로 업데이트 됨
오류가 날 가능성이 있음 (스터디를 통해 같이 수정해 나감)
```
